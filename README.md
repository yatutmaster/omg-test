# Lumen PHP Framework

* Сервис получения курсов валют в json
* GET запрос http://localhost:8080/currency/all
* Запуск синхронизации ежедневно

## Env dependencies

* Docker

## Local deploy

```bash
./local-deploy.sh build
```

## Testing

```bash
./vendor/bin/phpunit
```
